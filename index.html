¿Que es un Diagrama de Flujo?

Un diagrama de flujo es la representación gráfica de un algoritmo, proceso o sistema, que utiliza símbolos estandarizados conectados por flechas para mostrar la secuencia lógica de pasos, decisiones y flujos de trabajo. Sirve para simplificar la comprensión de tareas complejas, mejorar la eficiencia y documentar procedimientos en diversos campos como programación, ingeniería y gestión de negocios.

Reglas de un Diagrama de Flujo

  • Sentido de flujo

    El diagrama debe leerse de arriba hacia abajo y de izquierda a derecha.

  • Líneas de flujo

    Se deben usar líneas rectas, ya sean horizontales o verticales, evitando líneas diagonales.

  • Dirección

    Las líneas de flujo deben tener flechas en la punta para indicar la dirección del proceso.

  • Inicio y Fin

    Todo diagrama debe tener un inicio y un final claros.

  • No cruzar líneas

    Se debe evitar el cruce de líneas; si es necesario, se utilizan conectores para separar el flujo.

  • Decisiones

    Solo los símbolos de decisión (rombos) pueden tener más de una línea de salida (por ejemplo, sí/no, verdadero/falso).

  • Conexiones

    Las líneas de flujo no deben quedar sin conectar.

  • Texto

    El texto dentro de los símbolos debe ser breve, preciso y legible.

  • Uso de símbolos

    Se deben usar los símbolos estándar (óvalos para inicio/fin, rectángulos para procesos, rombos para decisiones).

Simbolos de los Diagramas de Flujo

Terminación

Terminación se refiere al inicio o fin de un proceso. Se representa con un símbolo especial: un óvalo (o a veces un rectángulo redondeado), a menudo etiquetado con palabras como "Inicio", "Fin" o "Salida".

Proceso

Un proceso representa una tarea, acción u operación específica que necesita realizarse. Es uno de los elementos más comunes en los diagramas de flujo y se muestra típicamente como un rectángulo.

Datos

El símbolo de Datos se utiliza para representar información que se almacena o recupera, típicamente de un archivo, base de datos u otro medio de almacenamiento. Generalmente se dibuja como un rectángulo inclinado.

Decisión

Una decisión representa un punto donde se debe tomar una elección, típicamente una pregunta de sí/no o verdadero/falso. Se simboliza con una forma de diamante

Documento

El símbolo de Documento representa un documento único generado, recibido o utilizado en un proceso. Parece un rectángulo con un borde inferior ondulado (curvado), asemejándose a una hoja de papel.

Preparación

Se utiliza para indicar un paso de configuración inicial, ajuste de variables, o preparación antes de que comience el proceso principal.

Flecha

Las flechas en los diagramas de flujo juegan un papel crucial: muestran la dirección del flujo, guiando al lector a través del proceso de un paso al siguiente.

Conector

El símbolo de Conector se utiliza para enlazar diferentes partes de un diagrama de flujo, especialmente cuando el diagrama es demasiado grande para caber en una página. Se representa por un pequeño círculo con una etiqueta dentro.

Conector a otra página

El símbolo de Conector Fuera de Página se utiliza para mostrar que el flujo continúa en una página o sección diferente del diagrama de flujo. Tiene forma de pentágono (a veces llamado forma de "home plate").
